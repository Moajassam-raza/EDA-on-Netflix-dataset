Netflix Dataset -- Exploratory Data Analysis (EDA)

This project performs a complete Exploratory Data Analysis (EDA) on
the Netflix Movies and TV Shows Dataset.
The goal is to uncover meaningful insights about Netflix's content
library, including trends in genres, release years, ratings, countries,
and overall content distribution.

Project Overview

The Netflix dataset contains detailed information about movies and TV
shows available on the platform like kaggle.
Through this EDA project, we explore:

-   Content availability trends
-   Genre popularity
-   Movie vs.Â TV show distribution
-   Content ratings and audience targeting
-   Country-wise content production
-   Year-wise content addition
-   Duration patterns
-   Relationships between multiple features

Dataset Information

-   Dataset Name:Netflix Movies & TV Shows
-   Source: Kaggle
